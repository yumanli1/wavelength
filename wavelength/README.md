# Wavelength Game

## Overview
Wavelength is a two-player game where players take turns giving hints and guessing based on those hints. The game board is designed as a half-circle, creating a unique and engaging gameplay experience.

## Features
- Two-player gameplay
- Hint submission by the current player
- Guess submission by the opposing player
- Visual representation of the game state on a half-circle board
- Player score tracking and turn management

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node package manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/wavelength-game.git
   ```
2. Navigate to the project directory:
   ```
   cd wavelength-game
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Game
To start the game, run the following command:
```
npm start
```
This will launch the application in your default web browser.

## Gameplay Rules
1. Players take turns. The current player provides a hint related to a specific concept or word.
2. The opposing player must guess the concept based on the hint provided.
3. The game continues until a predetermined number of rounds or until a player reaches a specific score.
